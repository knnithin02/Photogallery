📸 Tourism Photo Gallery Website

A simple and visually appealing static website showcasing tourism destinations through a photo gallery. This project highlights beautiful travel locations and provides an engaging browsing experience using HTML and CSS.

🌍 About the Project

The Tourism Photo Gallery Website is designed to display images of popular travel destinations in an organized and attractive layout. It is built as a static website, making it lightweight, fast, and easy to deploy.

This project is ideal for beginners learning front-end development and practicing layout design using HTML and CSS.

✨ Features

📷 Clean and responsive photo gallery layout
🌐 Static website (no backend required)
⚡ Fast loading and lightweight
🎨 Styled using CSS for better visual appeal
🧭 Easy navigation and simple design

🛠️ Tech Stack

HTML5
CSS3
